######################
Pypeline documentation
######################

*Pypeline* is a `GPLv3 <https://www.gnu.org/licenses/gpl-3.0.en.html>`_ signal processing toolkit to
design and deploy holistic imaging pipelines.  It is designed around the concept of
:py:class:`~pypeline.core.Block`: callable objects that implement specific actions.
